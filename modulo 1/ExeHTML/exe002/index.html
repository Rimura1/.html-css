Pricipais formatações

Negrito / Destaque

Para fazer um texto ficar em negrito, basta usar a tag <b> (não semantica/ sem significado)

para fazer um texto ficar em negrito, basta usar a tag <strong>(semantica / com significadoHTML5)

Italico / Enfase

Para fazer um texto ficar em italico, basta utilizar a tag <i> (não semantica)

Para fazer um texto ficar em italico, basta utilizar a tag <em> (semantica)

Texto inserido

Para marcar um texto inserido para dar destaque de que ele foi colocado depois (semantico)

Tambem podemos usar o sublinhado com a tag <u> (não semantica)

Texto Grande/Pequeno

Estamos criando um texto grande usando a tag <Big> apesar de ela estar obsoleta (não semantica),
para criar um texto pequeno, usamos a tag <small>

Texto marcado

Para marcar um texto, assim, basta utilizar a tag <mark>

Texto cancelado

Para marcar um texto como cancelado usamos a tag <del>

Textos Sobrescrito/Subescrito

Textos Sob e sub são para formulas por exemplo x=a+b2

x=a+b2 para cima usamos a tag <sup>

x=a+b2 para baixo usamos a tag <sub>
